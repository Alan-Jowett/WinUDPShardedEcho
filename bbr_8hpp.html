Detailed Description

Simple header-only BBR-like congestion controller (lightweight, approximate)

This file defines a basic BBR-like congestion controller that can be plugged into the client pacer. It estimates bandwidth and RTT to adjust the sending rate. The implementation is simplified for demonstration purposes and may not include all features of a full BBR implementation.
