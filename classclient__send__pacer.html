Detailed Description

template<class CongestionController = null_congestion_controller>
class client_send_pacer< CongestionController >

Client send pacer using token bucket algorithm.

Note
: Not thread-safe; caller must ensure synchronization if used from multiple threads.

Query whether a packet can be sent now.

This method obtains the current steady-clock timestamp internally and checks the token bucket state.

Returns
true A packet may be sent immediately.
