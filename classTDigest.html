Detailed Description

Mergeable t-digest implementation without internal locking.

TDigest is intended for per-CPU or per-thread usage where each partition accumulates values locally via add() and later the digests are merged using merge() to compute global percentiles. The implementation buffers raw points and compresses them into centroids when compress() or merge() is called.
