Detailed Description

Simple command-line option registrar and parser.

Usage example:

ArgParser p;
p.add_option("server", 's', "localhost", true, "server address");
p.parse(argc, argv);
auto host = p.get("server");

Parse command-line arguments.

Unknown options are silently ignored. Positional arguments are not stored by this parser. Supports GNU-style --name=value, --name value and single-character short flags -s value or -f (for non-value flags).
