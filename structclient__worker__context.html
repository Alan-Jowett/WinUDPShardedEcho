Detailed Description

Per-worker context holding sockets, IOCP and statistics.

Each worker owns one IOCP and one or more UDP sockets affinitized to the worker's processor. The struct tracks per-worker counters for sent/recv packets, outstanding sequence numbers, and RTT aggregates.
